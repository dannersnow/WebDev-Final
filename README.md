# WebDev-Final

The site is deployed at: https://dannersnow.github.io/WebDev-Final/

This is my final project for CS463 - Intro to WebDev. It is a personal website broken into different sections that showcase different web development concepts in html, css, and javascript.

Running the site is pretty self explanatory, and if there is anything that could be missed, there are directions in that section. Some useful elements I used were buttons, fieldsets, cards, and containers. I tried to make everything look cohesive and have a general theme.

I used our previous lab exercises for most of the help on my code, but also used W3Schools quite a bit for specific questions, especially with javascript.
